<details>
<summary><b>📖 Why this order?</b></summary>
<br/>

- **Docs first** — get the accurate, unfiltered mental model straight from the source before anyone else's interpretation of it.
- **YouTube tutorials** — layer in the visual, hands-on walkthroughs to see the theory applied.
- **Structured courses** (Udemy, freeCodeCamp) — patch the gaps and get a proper end-to-end curriculum.
- **Ship a real project** — nothing sticks until it's been built, broken, and fixed under real conditions.

## 🚀 Projects

<div align="center">

[![Nmap-NSE-Script-Collection](https://github-readme-stats.vercel.app/api/pin/?username=NexzaDev&repo=Nmap-NSE-Script-Collection&theme=chartreuse-dark&hide_border=true&bg_color=0D1117&title_color=00FF9C&text_color=c9d1d9&icon_color=1E90FF)](https://github.com/NexzaDev/Nmap-NSE-Script-Collection)

**Custom Nmap NSE scripts for defensive, informational security audits** — a 55+ script collection spanning HTTP, DNS, SMB, SSL/TLS, database, and FTP services. Each script targets a specific, real-world misconfiguration: missing security headers, insecure cookie flags, open DNS resolvers, weak TLS ciphers, unauthenticated database access, anonymous FTP uploads, and dozens more. Built to run standalone or combined into full infrastructure-wide audit scans, with a strong emphasis on safe, non-destructive checks that are appropriate for production environments.
